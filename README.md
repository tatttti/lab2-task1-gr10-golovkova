# Overview

This project contains the solution for Laboratory Work 2, Task 1 (Variant 40). The program processes a string with words separated by commas and spaces and outputs these words in reverse order.

# Usage

The project was completed by **Golovkova Tatyana, 10a group** and the program processes an input string provided as a command‑line argument and outputs the words in reverse order.

# Building

The project is developed and run in the Replit online environment. The .replit file contains the configuration required for compiling the main.c source file and launching the resulting program according to the assignment requirements.
